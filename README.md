# G13Exam

Restaurant API uchun ASP.NET Core Web API loyiha bazasi.

## Boshlang'ich Tuzilma

Loyihada quyidagi asosiy papkalar mavjud:

- Configurations
- Controllers
- Data
- Dtos
- Entities
- FluentApies
- Logs
- Mappings
- Migrations
- Repositories
- Services

## Talablar

- .NET 8 SDK
- SQL Server

## Ishga Tushirish

1. Restore qiling:

	`dotnet restore G13Exam/G13Exam.slnx`

2. Build qiling:

	`dotnet build G13Exam/G13Exam.slnx`

3. API ni ishga tushiring:

	`dotnet run --project G13Exam/Exam.Api/Exam.Api.csproj`

4. Swagger ni brauzerda oching:

	`https://localhost:<port>/swagger`

## Eslatma

- Solution fayli Git repository bilan birga sozlangan.
- Swagger Development muhitida yoqilgan.
- Build muvaffaqiyatli o'tishi kerak.
